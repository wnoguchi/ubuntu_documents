# Ubuntuの自動インストール

刺し身タンポポ作業の自動化。  
CentOSならKickstart使うことになるけど、Ubuntuの場合はPreseedingというツールを使うらしい。

## 環境

- 母艦: Ubuntu 13.04 Desktop
- ISO化するもの: Ubuntu Server 12.04 LTS
